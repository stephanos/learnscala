Variablen

@comp.snippet { |42 } @comp.console { |res0: Int = 42 }
  • res0 ist der Name der Variable
    (fortlaufend nummeriert)
  • Int ist Typ der Variable
    (automatisch ermittelt)
  • 42 ist Wert der Variable
    (via toString())

Variablen

@comp.snippet { |42.toString() } @comp.console { |res0: java.lang.String = 42 }
@comp.snippet { |42.isWhole() } @comp.console { |res1: Boolean = true }
@comp.snippet { |42.==(42) } @comp.console { |res2: Boolean = true }
@comp.infobox() { Alles ist ein Objekt. Alles! }

Variablen

@comp.snippet { |val meaningOfLife: Int = 42 } @comp.console { |meaningOfLife: Int = 42 }
  • val beginnt Variable
  • meaningOfLife ist der Name
  • : trennt Name und Typ
  • Int ist der Typ
  • = weist Wert zu
  • 42 ist der Wert
@comp.infobox() { val leitet eine Variable ein, der Typ folgt hinter dem Namen. }

@comp.snippet { |val meaningOfLife: Int = 42 | |meaningOfLife = 101 } @comp.console { |meaningOfLife: Int = 42 | |:8: error: reassignment to val | meaningOfLife = 101 | ^ }
@comp.infobox() { val kann nicht überschrieben werden, sie ist unveränderlich (immutable). }
