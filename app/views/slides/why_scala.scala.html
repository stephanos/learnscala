Scala

A scalable language

The name Scala stands for scalable language.

The language is designed to grow with the demands of its users. You can apply Scala to a wide range of programming tasks, from writing small scripts to building large systems.

Martin Odersky,
creator of Scala

Scala is ... a scripting language


  • interactive Read-Eval-Print-Loop (REPL)
  • no semicolons
    |println("Hello World")
  • type inference
    |val i = 1 + 2 * 3 // the type of i is Int |val s = "Scala rocks" // the type of s is String

Scala is ... object-oriented


  • classes
    |class Car |class MonsterTruck extends Car
  • every (!) value is an object
    |val meaning_of_life = 42 |val my_new_car = new MonsterTruck

Scala is ... functional


  • every block of code is an expression

Scala is ... interoperable


  • compiles to regular Java-compatible byte code
  • compatible with thousands of Java libraries

Scala is ... statically typed


Scala is ... composable


Scala is ... extensible


Scala is ... concise


Scala is ... fast


  • runs on the Java Virtual Machine
