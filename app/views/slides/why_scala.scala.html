Scala

A scalable language

The name Scala stands for
scalable language.

The language is designed to grow with the demands of its users. You can apply Scala to a wide range of programming tasks, from writing small scripts to building large systems.

Martin Odersky,
creator of Scala

Scala is ... a scripting language

  • interactive Read-Eval-Print-Loop (REPL)
  • no semicolons @comp.snippet { |println("Hello World") }
  • type inference @comp.snippet { |val i = 1 + 2 * 3 // the type of i is Int |val s = "Scala rocks" // the type of s is String }

Scala is ... object-oriented

  • every (!) value is an object (no Java primitives like 'int') @comp.snippet { |42.toFloat }
  • classes and inheritance @comp.snippet { |class MonsterTruck extends Car }
  • mixin-based composition @comp.snippet { |class FlyingCar extends Car with Wings }

Scala is ... functional

  • re-usable blocks of code that have no side effects @comp.snippet { |val names = users.map(_.username) }
    @comp.snippet { |val minors = users.map(_.age()).filter(_ < 18) }
  • pattern matching @comp.snippet { |def matchTest(arg: String, value: Any) = (arg, value) match { | case ("-h" | "--help", _) => displayHelp() | case ("-v" | "--version", _) => displayVersion() | case (a,v) => sys.error("unknown argument:" + a) |} }

Scala is ... Java

  • compiles to regular Java byte code
    • as fast as Java (sometimes faster!)
    • calling & extending Java class from Scala? - works perfectly
    • calling & extending Scala class from Java? - works mostly, a few exceptions


=> compatible with all the thousands of Java libraries - no glue code needed

Scala is ... statically typed

  • very advanced static type system
    • type inference @comp.snippet { |val i = 1 + 2 * 3 // the type of i is Int }
    • compound types @comp.snippet { |def cloneAndReset(obj: Cloneable with Resetable) = .. }
  • benefits: less mistakes, reduced testing and safe refactoring
  • drawbacks: increased compilation time

Scala is ... extensible

  • add your own control structures @comp.snippet { |def retry[T](n: Int)(fn: => T): T = | try { fn } | catch { case e => if (n > 1) retry(n - 1)(fn) else throw e } }
  • create a complete Domain Specific Language (DSL) @comp.snippet { @Html(""" |class HelloWorldSpec extends Specification { | "The 'Hello world' string" should { | "contain 11 characters" in { | "Hello world" must have size(11) | } """) }

Scala is ... concise

  • reduces lines of code by at least 50%
    • less boilerplate @comp.snippet { |class MyCar(wheels: Int = 4) }
    • factor out common behaviour by using traits and closures @comp.snippet { |trait Logging |def getAttributeFromAllUsers[T](fn: (User) => List[T]) }
  • quicker to write , easier to read and less error prone
