Scala, die unbekannte Sprache

Der Name Scala steht für
scalable language.

Die Sprache ist so konzipiert, dass sie mit den Anforderungen des Nutzers wächst. Man kann sie auf eine Vielzahl von Aufgaben anwenden, von kleinen Skripten bis hin zu großen System.

Martin Odersky,
Erfinder von Scala
