Scala, die unbekannte Spache

Der Name Scala steht für
scalable language.

Die Sprache ist so konzipiert, dass sie mit den Anforderungen des Nutzers wächst. Man kann sie auf eine Vielzahl von Aufgaben anwenden, von kleinen Skripten bis hin zu großen System.

Martin Odersky,
Erfinder von Scala

Scala ist ... eine Skriptsprache

  • interaktive Konsole: Read-Eval-Print-Loop (REPL)
  • keine Semikolons @comp.snippet { |println("Hello World") }
  • Type Inference @comp.snippet { |val i = 1 + 2 * 3 // the type of i is Int |val s = "Scala rocks" // the type of s is String }

Scala ist ... objektorientiert

  • jeder (!) Wert ist ein Objekt (keine primitiven Typen wie 'int') @comp.snippet { |42.toFloat }
  • Klassen und Vererbung @comp.snippet { |class MonsterTruck extends Car }
  • Mixin-Mechanismus @comp.snippet { |class FlyingCar extends Car with Wings }

Scala ist ... knapp und präzise

  • reduziert die Code-Zeilen um mehr als 50%
    • weniger Boilerplate-Code @comp.snippet { |class MyCar(wheels: Int = 4) }
    • Traits und Closures für Wiederverwendung @comp.snippet { |trait Logging |def getAttributeFromAllUsers[T](fn: (User) => List[T]) }
  • schneller zu schreiben und leichter zu lesen

Scala ist ... funktional

  • wiederverwendbare Code-Blöcke ohne Seiteneffekte @comp.snippet { |val names = users.map(_.username) }
    @comp.snippet { |val minors = users.map(_.age()).filter(_ < 18) }
  • Pattern Matching @comp.snippet { |def matchTest(arg: String, value: Any) = (arg, value) match { | case ("-h" | "--help", _) => displayHelp() | case ("-v" | "--version", _) => displayVersion() | case (a,v) => sys.error("unknown argument:" + a) |} }

Scala ist ... statisch typisiert

  • sehr fortschrittliches Typenmodell
    • Type Inference @comp.snippet { |val i = 1 + 2 * 3 // the type of i is Int }
    • Compound Types @comp.snippet { |def cloneAndReset(obj: Cloneable with Resetable) = .. }
  • Vorteil: weniger Flüchtigkeitsfehler & Testaufwand sowie sicheres Refactoring
  • Nachteil: erhöhte Kompilierzeit

Scala ist ... Java

  • erstellt gewohntem Java byte code
    • so schnell wie Java (manchmal schneller!)
    • Aufrufe & Erweiterung einer Java-Klasse von Scala? - reibungslos
    • Aufrufe & Erweiterung einer Scala-Klasse von Java? - geht, mit Ausnahmen


⇨ kompatibel mit den tausenden Java-Bibliotheken - ohne zusätzlichen Code

Scala ist ... erweiterbar

  • eigene Kontrollstrukturen @comp.snippet { |def retry[T](n: Int)(fn: => T): T = | try { fn } | catch { case e => if (n > 1) retry(n - 1)(fn) else throw e } }
  • eigene Sprachen: Domain Specific Language (DSL) @comp.snippet { @Html(""" |class HelloWorldSpec extends Specification { | "The 'Hello world' string" should { | "contain 11 characters" in { | "Hello world" must have size(11) | } """) }
