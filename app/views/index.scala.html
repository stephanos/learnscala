Scala ist das neue Java: Eine moderne, typensichere Sprache, die objektorientierte und funktionale Programming auf der bekannten Java-Plattform vereint und sich doch wie eine dynamische Sprache anfühlt. Damit werden Programme deutlich kürzer, verständlicher, performanter und fehlerfreier - die Schulung hilft Ihnen diese Vorteile schnellstmöglich zu nutzen.

 
 

Manager ?

Scala ermöglicht es Entwicklern produktiver zu sein. Wie? Es reduziert die Code-Zeilen für Programme um 50%, behält dabei aber die Qualität und Performanz von Java.

Scala ist zwar eine eigene Sprache, läuft aber auf der Java-Plattform. Das heißt bestehende Bibliotheken und Programme können einfach weiterverwendet werden.

Scala wird von Startups wie Twitter und LinkedIn aber auch etablierten Firmen wie Guardian und Sony verwendet. Die Sprache ist in den letzten 9 Jahren zu einem stabilen und produktiven Werkzeug gereift - seit kurzem auch mit der speziell gegründeten Firma Typesafe im Rücken.

Entwickler ?

Scala ist eine Revolution. Es mutet an wie eine dynamische Sprache a la Groovy, doch behält die volle Typensicherheit und Performanz die man von Java gewohnt ist.

Viele wertvolle Features wie closures, type inference, traits und case classes machen Programme in Scala kürzer, lesbarer und sicherer als in Java - und auch noch Spaß.

Aber durch diesen Djungel an neuen Möglichkeiten muss erst einmal durchgefunden werden. Dieser Kurs hilft bei einem zielführenden Start mit Scala - doch seien Sie gewarnt, danach wird Ihnen Java-Code wie aus einer antiken Zeit vorkommen.
