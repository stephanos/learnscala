Scala ermöglicht es Entwicklern produktiver zu sein. Wie? Programmcode wird durchschnittlich um über 50% reduziert, behält dabei aber die Performanz und Qualität von Java.

Scala ist zwar eine eigene Sprache, weil es aber auf der Java-Plattform aufsetzt, können Sie bestehenden Java-Code und bekannte Bibliotheken weiterverwenden.

Scala wird von Startups wie Twitter und LinkedIn aber auch etablierten Firmen wie Guardian und Sony verwendet. Die Sprache ist in den letzten 9 Jahren zu einem stabilen und produktiven Werkzeug gereift - inklusive der kommerziellen Firma 'Typesafe' im Rücken.

Scala ist eine Revolution. Es mutet an wie eine dynamische Sprache a la Groovy, doch behält die Performanz und Typensicherheit die man von Java gewohnt ist.

Viele wertvolle Features wie closures, type inference, traits und case classes machen Programme kürzer, lesbarer und sicherer als in Java - und auch noch Spaß.
