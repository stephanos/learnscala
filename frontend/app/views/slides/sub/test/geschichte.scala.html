Martin Odersky
Der Erfinder von Scala
  • geboren 1958
  • Doktor an der ETH Zürich
  • Pizza-Sprache:
    Grundlage für Java Generics und javac
  • 1999: Professor an der EPF de Lausanne
  • arbeitet seit 2001 an Scala
  • erste Release 2003
  • 2.3€ Mio. Forschungsgelder von der EU
  • gründete Typesafe für Support von Scala
