@Images.types(0) @fragment {

Typ 'Any'

  • gemeinsame Super-Klasse aller Typen
  • finale Methoden:
    • ==
    • !=
    • isInstanceOf
    • asInstanceOf
  • Standardimplementierungen für:
    • equals
    • hashCode
    • toString
}
} @slide(topic = topic, title = "AnyVal") {
@Images.types(1) @fragment {

Typ 'AnyVal'

  • "Value Types"
  • kann nicht direkt vererbt (final) oder instanziiert (abstrakt) werden
  • entspricht primitiven Datentypen aus Java (werden im Hintergrund auch verwendet)
  • Ausnahme ist Scala-spezifischer Typ Unit (wie void)
}
} @slide(topic = topic, title = "AnyRef") {
@Images.types(2) @fragment {

Typ 'AnyRef'

  • "Reference Types"
  • entspricht java.lang.Object in Java bzw. System.Object in .Net
  • vererbt an Java-Klassen (z.B. java.lang.String) und Scala-Klassen (z.B. scala.Predef)
  • zusätzlich erben Scala-Klassen vom Trait ScalaObject (einzig für interne Zwecke)
}
} @slide(topic = topic, title = "Null") {
@Images.types(3) @fragment {

Typ 'Null'

  • "Bottom Type"
  • Unterklasse von allen Reference Types
  • einziger möglicher Wert ist null
}
