@fragment {

Typ 'Any'

  • gemeinsame Super-Klasse aller Typen
  • finale Methoden:
    • ==
    • !=
    • isInstanceOf
    • asInstanceOf
  • Standardimplementierungen für:
    • equals
    • hashCode
    • toString
}
} @slide() { } {
@fragment {

Typ 'AnyVal'

  • "Value Types"
  • kann nicht direkt verebt (final) oder instanziiert (abstrakt) werden
  • entspricht primitiven Datentypen aus Java (werden im Hintergrund auch verwendet)
  • Ausnahme ist Scala-spezifischer Typ Unit (wie void)
}
} @slide() { } {
@fragment {

Typ 'AnyRef'

  • "Reference Types"
  • entspricht java.lang.Object
  • vererbt an Java-Klassen (z.B. java.lang.String) und Scala-Klassen (z.B. scala.Predef)
  • zusätzlich erben Scala-Klassen vom Trait ScalaObject (einzig für interene Zwecke)
}
} @slide() { } {
@fragment {

Typ 'Null'

  • "Bottom Type"
  • Unterklasse von allen Reference Types
  • einziger möglicher Wert ist null

Typ 'Nothing'

  • "Bottom Type"
  • Unterklasse aller Typen
  • hat keinen Wert, wird aber z.B. bei Exceptions benötigt:
