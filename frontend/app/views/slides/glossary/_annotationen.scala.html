@layout.simpledeck("Annotationen") {
  • fügen Metadaten zu Klassen, Methoden, Feldern und Parametern hinzu - sparsam einsetzen!
  • alle Java-Annotationen können verwendet werden, wichtige Scala-spezifische Annotationen sind:
    @sidebyside() {
    @@BeanProperty
     Feld erhält get- und set-Methoden
    @@deprecated
     Compiler-Warnung bei Verwendung
    @@elidable
     Code kann via Compiler-Flag entfernt werden
    @@native
     Methode ist in C/C++ implementiert
    @@tailrec
     Methode muss Tail Rekursive sein
    } {
    @@throws
     Methode erhält Checked Exception-Signatur
    @@transient
     Feld wird nicht serialisiert
    @@unchecked
     underdrückt Warnungen beim Pattern Matching
    @@varargs
     Java-kompatible Hilfsmethode fürs varargs
    @@volatile
     Feld kann aus mehreren Threads gelesen werden
    }
}
