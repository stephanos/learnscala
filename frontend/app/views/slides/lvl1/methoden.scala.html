Methoden

@headbody() { Methoden: Definition } {
@snippet() { @sourceblock(fragment = false) { |def hello(): String = { | return "Hello World" |} } @outblock() { |hello: String } }
  • def startet die Definition
  • hello ist der Name der Methode
  • () beinhaltet die Liste der Parameter
  • String ist der Rückgabetyp
  • { .. } enthält Implementierung
@infobox() { def leitet eine Methode ein - diese hat immer einen Rückgabetyp. } }
Difference between method and function in Scala
@headbody() { Methoden: Unit } {
@snippet() { @sourceblock(fragment = false) { |def hello() = { | println("Hello World") |} } @outblock() { |hello: Unit } } @snippet() { @sourceblock() { |def hello() { | println("Hello World") |} } @outblock() { |hello: Unit } }
@infobox() { Hat eine Methode keinen Rückgabewert, hat sie den Typ Unit. } }
@headbody() { Methoden: Optionales } {
@sourcecode(id = "hello") { |def hello(): String = | return "Hello World" }
ohne geschweifte Klammern
Wann? Nur eine Zeile
@sourcecode() { |def hello(): String = | "Hello World" }
ohne return
Wann? Immer, Rückgabe ist Ergebnis des letzten Ausdrucks
@sourcecode() { |def hello: String = | "Hello World" }
ohne Parameterliste
Wann? Keine Parameter
@sourcecode() { |def hello = | "Hello World" }
ohne Rückgabetyp
Wann? Nicht rekursiv oder überladen
@sourcecode() { |def hello = "Hello World" }
ohne Zeilenumbruch
