Methoden

und Ihre Definition

@comp.headbody() { Methoden: Definition } {
@comp.snippet() { @comp.sourceblock() { |def hello(): String = { | return "Hello World" |} } @comp.outblock() { |hello: String } }
  • def startet die Definition
  • hello ist der Name der Methode
  • () beinhaltet die Liste der Parameter
  • String ist der Rückgabetyp
  • { .. } enthält Implementierung
@comp.infobox() { def leitet eine Methode ein - diese hat immer einen Rückgabetyp. } }
Difference between method and function in Scala

@comp.infobox() { Hat eine Methode keinen Rückgabewert, hat sie den Typ Unit. } }

ohne return

ohne Parameterliste

ohne Rückgabetyp
Wann? Nicht rekursiv oder überladen
@comp.sourcecode() { |def hello = "Hello World" }
