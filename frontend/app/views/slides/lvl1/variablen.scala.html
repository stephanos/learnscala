@layout.slide("Variablen") {

Variablen

@headbody() { Variablen } {
@snippet() { @callblock(fragment = false) { |42 } @outblock() { |res0: Int = 42 } }
  • res0 ist der Name der Variable
    (fortlaufend nummeriert)
  • Int ist Typ der Variable
    (automatisch ermittelt)
  • 42 ist Wert der Variable
    (via toString())
}
@headbody() { Variablen: Objekte } {
@snippet() { @callblock(fragment = false) { |42.toString() } @outblock() { |res0: java.lang.String = 42 } }
Methode toString von Objekt 42
@snippet() { @callblock(fragment = false) { |42.isWhole() } @outblock() { |res1: Boolean = true } }
Methode isWhole
@snippet() { @callblock(fragment = false) { |42.==(42) } @outblock() { |res2: Boolean = true } }
Methode ==
@infobox() { Alles ist ein Objekt und jedes Objekt kann Operatoren als Methoden haben. } }
@headbody() { Variablen: Definition } {
@snippet() { @callblock(fragment = false) { |val meaningOfLife: Int = 42 } @outblock() { |meaningOfLife: Int = 42 } }
  • val beginnt Variable
  • meaningOfLife ist der Name
  • : trennt Name und Typ
  • Int ist der Typ
  • = weist Wert zu
  • 42 ist der Wert
@infobox() { val leitet eine Variable ein, der Typ folgt hinter dem Namen. } }
@headbody() { Variablen: Immutable } {
@snippet() { @callblock(fragment = false) { |val meaningOfLife: Int = 42 } @outblock() { |meaningOfLife: Int = 42 } @callblock() { |meaningOfLife = 101 } @outblock() { |:8: error: reassignment to val | meaningOfLife = 101 | ^ } }
meaningOfLife wird mit val definiert und versucht zu überschreiben.
@infobox() { val kann nicht überschrieben werden, sie ist unveränderlich (immutable). } }
@headbody() { Variablen: Typensicherheit } {
@snippet() { @callblock(fragment = false) { |val meaningOfLife: Int = "Java" } @outblock() { |:7: error: type mismatch; |found : java.lang.String("Java") |required: Int | val meaningOfLife: Int = "Java" | ^ } }
meaningOfLife wird als Int definiert aber mit dem String Java initialisiert
@infobox() { Der Compiler stellt sicher, dass Variablentyp und -wert zusammenpassen. } }
@headbody() { Variablen: Optionales } {
@snippet() { @callblock(fragment = false) { |val meaningOfLife = 42 } @outblock() { |meaningOfLife: Int = 42 } }
meaningOfLife wird mit dem Wert 42 initialisiert, der Typ Int wird zugewiesen
@snippet() { @callblock(fragment = false) { |val pi = 3.14f } @outblock() { |pi: Float = 3.14 } }
pi wird mit dem Wert 3.14f initialisiert, der Compiler erkennt den Typ Float
@infobox() { Der Variablentyp kann weggelassen werden, der Compiler leitet ihn ab. } }
@headbody() { Variablen: Mutable } {
@snippet() { @callblock(fragment = false) { |var fuel = 100.0f } @outblock() { |fuel: Float = 100.0 } @callblock() { |fuel = 99.5 } @outblock() { |fuel: Float = 99.5 } }
Variable fuel wird initialisiert mit 100%
Der Wert wird mit einer weiterer Anweisung auf 99.5% überschrieben
@infobox() { var kann überschrieben werden, sie ist veränderlich (mutable). } @warnbox() { var sollte vermieden und sehr bedacht eingesetzt werden. } }
