Variablen

@comp.headbody() { Variablen } {
@comp.snippet() { @comp.callblock() { |42 } @comp.outblock() { |res0: Int = 42 } }
  • res0 ist der Name der Variable
    (fortlaufend nummeriert)
  • Int ist Typ der Variable
    (automatisch ermittelt)
  • 42 ist Wert der Variable
    (via toString())
}

@comp.headbody() { Variablen: Definition } {
@comp.snippet() { @comp.callblock() { |val meaningOfLife: Int = 42 } @comp.outblock() { |meaningOfLife: Int = 42 } }
  • val beginnt Variable
  • meaningOfLife ist der Name
  • : trennt Name und Typ
  • Int ist der Typ
  • = weist Wert zu
  • 42 ist der Wert
@comp.infobox() { val leitet eine Variable ein, der Typ folgt hinter dem Namen. } }
@comp.headbody() { Variablen: Immutable } {
@comp.snippet() { @comp.callblock() { |val meaningOfLife: Int = 42 } @comp.outblock() { |meaningOfLife: Int = 42 } @comp.callblock() { |meaningOfLife = 101 } @comp.outblock() { |:8: error: reassignment to val | meaningOfLife = 101 | ^ } }
meaningOfLife wird mit val definiert und versucht zu überschreiben.
@comp.infobox() { val kann nicht überschrieben werden, sie ist unveränderlich (immutable). } }
@comp.headbody() { Variablen: Typensicherheit } {
@comp.snippet() { @comp.callblock() { |val meaningOfLife: Int = "Java" } @comp.outblock() { |:7: error: type mismatch; |found : java.lang.String("Java") |required: Int | val meaningOfLife: Int = "Java" | ^ } }
meaningOfLife wird als Int definiert aber mit dem String Java initialisiert
@comp.infobox() { Der Compiler stellt sicher, dass Variablentyp und -wert zusammenpassen. } }
@comp.headbody() { Variablen: Optionales } {
@comp.snippet() { @comp.callblock() { |val meaningOfLife = 42 } @comp.outblock() { |meaningOfLife: Int = 42 } }
meaningOfLife wird mit dem Wert 42 initialisiert, der Typ Int wird zugewiesen
@comp.snippet() { @comp.callblock() { |val pi = 3.14f } @comp.outblock() { |pi: Float = 3.14 } }
pi wird mit dem Wert 3.14f initialisiert, der Compiler erkennt den Typ Float
@comp.infobox() { Der Variablentyp kann weggelassen werden, der Compiler leitet ihn ab. } }
@comp.headbody() { Variablen: Mutable } {
@comp.snippet() { @comp.callblock() { |var pi = 3.14f } @comp.outblock() { |pi: Float = 3.14 } @comp.callblock() { |pi = 3.0f } @comp.outblock() { |pi: Float = 3.0 } }
 
@comp.infobox() { var kann überschrieben werden, sie ist veränderlich (mutable). } @comp.warnbox() { var sollte vermieden und nur in Ausnahmefällen eingesetzt werden. } }
